## 🎯 Overview
SkillSwap is a full-stack MERN app that empowers users to exchange skills through scheduled sessions. Learn new skills or teach your expertise seamlessly. 

---

## 🌟 Features

- 🛠 **Skill Management**
  - ➕ Add, edit, view, and delete skills
  - 👨‍🏫 Assign teachers to skills

- 📅 **Session Booking**
  - 📖 Book and cancel skill sessions
  - 👥 View all booked sessions

- 🧭 **Navigation**
  - 🔀 Multi-page SPA with React Router

- 🎨 **UI/UX**
  - 💎 Responsive design with fixed, full-screen background image
  - ⚡ Smooth navigation and instant updates

---

## 🛠 Technologies Used

| Layer       | Technology           |
|-------------|---------------------|
| Frontend    | React.js, Vite      |
| Routing     | React Router DOM    |
| Backend     | Node.js, Express    |
| Database    | MongoDB            |
| HTTP Client | Axios               |
| Styling     | CSS (External + Inline) |

---
## Usage

- Navigate between skills and sessions using the navigation links.
- Add new skills and sessions through dedicated forms.
- Edit or cancel existing skills and sessions.
- Enjoy the fixed background image enhancing the user experience.
